# まなびより 使い方メモ

## 通常メモの書き方

1. ホーム画面または下部ナビからメモ登録画面を開きます。
2. 分野、種別、タイトル、日付を入力します。
3. 出来事、気づき、教訓を分けて書きます。
4. 迷ったこと、比較した選択肢、判断基準があれば残します。
5. 次の行動を1つ書きます。

種別と感情は複数選択できます。種別は最低1つ選んでください。

## 振り返り画面の使い方

週・月・年のタブで期間を切り替えます。

確認できる内容:

- 期間内のメモ一覧
- 種別ごとの件数
- よく使ったタグ
- 未完了の次の行動
- 自分で書く振り返りコメント

週の開始日は設定画面で月曜日または日曜日に変更できます。

## 人物プロフィールの使い方

人一覧から人物プロフィールを登録します。

残せる内容:

- 名前、呼び名、関係性
- 初めて会った日、出会った場所
- 趣味、好きなもの
- 苦手そうなもの・配慮したいこと
- 大事にしていそうなこと
- 次に話したいこと

プロフィールは後から編集・削除できます。

## 会話メモの使い方

人物詳細画面から「会話メモを追加」を押します。

会った日ごとに、以下を記録できます。

- 会った日
- 会った場所
- 話した内容
- 印象に残ったこと
- 次に聞きたいこと
- 約束したこと
- フォローすること
- フォロー完了

同じ人物に複数の会話メモを追加できます。1件だけ削除しても、他の履歴は残ります。

## 1週間の試用で確認するポイント

- 通常メモを無理なく書けるか
- 種別や感情の選択肢が足りているか
- 週次振り返りで次の行動が見つけやすいか
- 人物プロフィールが次の会話に役立つか
- 会話メモの項目が多すぎないか
- 削除や編集の導線で迷わないか
